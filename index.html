Projects

Research projects from algorithm optimization to silicon implementation.

01

Motion Planning Accelerator for High DoF Manipulators

2025 - Present | Seoul National University

Hardware-friendly RRT-Connect optimization and end-to-end accelerator implementation for high-DoF manipulators. Taped out using Samsung 28nm FDSOI.

Role: Proposed the LIN RRT-Connect algorithm by replacing nearest-neighbor search with a Last Inserted Node strategy, designed a reconfigurable architecture for forward kinematics and sphere-based collision checking, and led the RTL-to-tape-out implementation flow.

Python, PyBullet, Verilog, VCS, Design Compiler, Innovus, PrimeTime, Virtuoso, Calibre

02

Physical Implementation of Log-Quantized GEMM Accelerator

2025 | Seoul National University

PPA optimization, synthesis, APR, and simulation for a log-quantized GEMM accelerator published in IEEE TCAD.

Role: Performed synthesis and place-and-route, optimized the physical implementation for PPA, and supported post-layout simulation and verification.

Verilog, VCS, Design Compiler, Innovus, PrimeTime, Virtuoso, Calibre

03

Reinforcement Learning Training Hardware for Autonomous Navigation

2022 - 2024 | SNU and NYCU

Deep reinforcement learning accelerator with hardware-friendly algorithm optimization, ASIC implementation, and real indoor verification on a ClearPath Jackal UGV. Taped out using Samsung 28nm LPP.

Role: Optimized the reinforcement learning algorithm, hardware architecture, and inference/training schedule; reduced neural-network parameters, external memory access, and computation; completed ASIC implementation, FPGA/PCB bring-up, and real-world robot verification.

ROS, Gazebo, Python, PyTorch, Verilog, VCS, ICC II, StarRC, PrimeTime, Vivado, KiCAD

04

FPGA-based Test Platform Design for Memristor Array Computing

2022 - 2024 | KAIST, SNU, SKKU, ETRI, Yonsei University

FPGA-based digital test platform for memristor array computing, including analog IP control and host-system communication.

Role: Designed the FPGA-based measurement platform, implemented digital control logic for analog IP operation, and built the host communication path used for array-computing experiments.

Verilog, AMD Vivado

Digital Test Block for CIM Macro

RTL implementation, synthesis, and APR for the digital test block of a compute-in-memory macro taped out using TSMC 28nm HPC+.

Verilog, VCS, Design Compiler, Innovus, StarRC, Virtuoso, Calibre
